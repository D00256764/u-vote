How It Works

For Organizers
  1. Create an election with options/candidates
  2. Upload voter email list (CSV)
  3. Generate secure one-time tokens
  4. Open the election
  5. Close and view results after voting ends
For Voters
  1. Receive unique voting token via email
  2. Access ballot using the token
  3. Cast your vote securely
  4. Receive confirmation hash
  5. Vote is anonymous and immutable
